# Exploratory-data-analysis-on-flight-data


## 📖 Overview

This project performs **Exploratory Data Analysis (EDA)** on a flight dataset to uncover patterns, trends, and key factors affecting flight delays. The dataset contains detailed information about flight schedules, delays, airlines, and routes.

The main goal is to analyze flight behavior and identify insights that can help understand delays and improve airline operations.

---

## 🎯 Objectives

* Understand the structure and features of the dataset
* Perform **Exploratory Data Analysis (EDA)**

  * Univariate Analysis
  * Bivariate Analysis
  * Multivariate Analysis
* Clean and preprocess the data
* Identify patterns in flight delays

---

## 📊 Dataset Description

The dataset contains **336,000+ flight records** with features such as:

* Flight date (year, month, day)
* Departure & arrival times
* Departure & arrival delays
* Airline carrier and flight number
* Origin and destination airports
* Air time and distance
* Scheduled vs actual timings

---

## 🔍 Exploratory Data Analysis

### 📌 Univariate Analysis

* Distribution of delays
* Flight frequency across months and days
* Airline distribution

### 📌 Bivariate Analysis

* Relationship between delay and airline carriers
* Delay vs distance
* Departure delay vs arrival delay

### 📌 Multivariate Analysis

* Combined impact of multiple features on delays
* Patterns across routes, carriers, and time

---

## ⚙️ Data Preprocessing

* Handling missing values
* Removing irrelevant features
* Outlier detection and treatment
* Encoding categorical variables
* Time feature transformation
* Feature scaling
* Handling skewed distributions

---

## 📈 Key Insights

* Flight delays are strongly influenced by **departure delays**
* Certain airlines and routes show higher delay patterns
* Time-related features (hour, month) impact delay trends
* Distance and air time correlate with flight performance
* Missing data exists in key delay-related columns
